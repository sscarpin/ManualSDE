    Asignar Áreas

    En esta pantalla se visualizarán todas las áreas asignadas a los usuarios, con sus correspondientes permisos.

    Nuevo

    En la esquina superior izquierda se verá el botón de alta, el cual sirve para generar una nueva asignación de área.

    Esto mostrará la pantalla para generar la nueva asignación de área.

    Tip

    Si en el formulario faltan datos obligatorios (*), el botón de guardado, se verá deshabilitado. Si se completan los datos requeridos el botón se verá habilitado y se podrá guardar el asignación de área.

    Filtros

    El usuario podrá buscar en todo el universo de asignar_asignar_areas del sistema y acceder a información de las mismas. Se verán los campos de búsqueda básica para buscar sobre la vista activa: nombre,...; los cuales pueden utilizarse individualmente o combinados.

    Info

    Los operadores a aplicar dependen del tipo de dato del campo de búsqueda y del contexto: texto (contiene, no contiene, igual, distinto), número (igual, distinto, contiene, no contiene), fecha (hasta, desde, entre, igual).

    Formulario de Asignar Áreas

    El usuario podrá acceder a las áreas asignadas a cada usuario.

    Cabecera del listado: cabecera con información sobre las columnas de ese listado.

    Listado: lista de los elementos de la vista activa, cada fila de la vista representa un asignación de área.

    Pie de listado: desde él se pueden acceder a las distintas páginas del mismo.

    Las opciones de acción que tienen los usuarios sobre las asignaciones de áreas son las siguientes:

    1- Modificar: mediante esta opción el usuario puede ingresar al formulario para modificar los datos de la asignación de áreas.

    Es este formulario se podrá modificar cualquiera de los datos de una asignación de área.

    Tip

    Si en el formulario faltan datos obligatorios (*), el botón de guardado, se verá deshabilitado. Si se completan los datos requeridos el botón se verá habilitado y se podrá guardar la asignación de área.

    2- Eliminar: mediante esta opción el usuario puede ingresar al formulario para eliminar la asignación de área.
